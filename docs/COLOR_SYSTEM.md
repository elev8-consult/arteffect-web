# ArtEffect Color System

## Primary Brand Colors
- Burgundy: `#5A0000`
- Gold: `#E6D3A3`
- Dark Background: `#111111`

## Neutral Colors
- Primary Text: `#F5F5F5`
- Secondary Text: `#BFBFBF`
- Card Background: `#161616`
- Borders: `#2A2A2A`

## Artistic Accent Colors
- Electric Blue: `#1DA1F2`
- Deep Ocean Blue: `#0D3B66`
- Sunflower Yellow: `#F2C94C`
- Terracotta Orange: `#E76F51`
- Forest Green: `#2A9D8F`
- Soft Lavender: `#A78BFA`
- Warm Coral: `#FF6B6B`

## Usage Rules
- Main background: `#111111`
- Hero/brand sections: `#5A0000`
- Cards: `#161616`
- Borders: `#2A2A2A`
- Primary button: `#E6D3A3` text on `#111111`
- Primary button hover: `#F2C94C`

## Accent Mapping
- Artist sections: `#1DA1F2`
- NGO highlights: `#2A9D8F`
- Design storytelling: `#F2C94C`
- Impact stats: `#FF6B6B`
- Batch highlights: `#A78BFA`

## Implementation Notes
- Keep the base UI dark and minimal so artwork remains the focal point.
- Apply accents as micro-highlights (eyebrows, badges, borders, hover states).
- Avoid full-page accent backgrounds except burgundy brand blocks.
